## Model
使用什么样的**映射函数**表示X和Y之间的关系

    - Linear Regression
     continuous → continuous
    - Logistic Regression
     continuous → 0/1
  - **Tree Model**
    - Decision Tree
    - Random Forest
  - **Neural Network Model**
    - Deep Learning
     不断更改参数，使得*Loss Function*越来越小
